## Dataset

This project uses the Credit Card Customer Churn dataset.

* Source: https://www.kaggle.com/datasets/rjmanoj/credit-card-customer-churn-prediction
* The dataset contains customer information such as age, income, credit limit, transaction history, etc.

### Key Details

* Around 10,000 customers
* ~18 features (demographics + financial behavior)
* Target: Predict whether a customer will churn (leave the service)

### Note

The dataset is not included in this repository due to size constraints.
Please download it manually from the above link.
